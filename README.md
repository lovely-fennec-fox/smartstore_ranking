# Demo
1) 내가 올린 쇼핑몰이 있다고 가정합니다.
![](/main/cap_01.gif)

2) 쇼핑몰에 주소값을 가져와서
![](/main/cap_02.gif)

3) 프로그램에 넣어주고 추가하면 해당 제품의 이름이 입력됩니다.
![](/main/cap_03.gif)

4) 해당 쇼핑몰을 찾는 키워드 값을 10개까지 넣을 수 있습니다.
![](/main/cap_04.gif)

5) 해당 키워드 값을 확인 한 후, 키워드에 따른 쇼핑몰에 보이는 순위를 검색하면 
![](/main/cap_05.gif)

6) 해당 키워드로 검색했을 때 노출되는 순위를 얻을 수 있습니다(파워랭킹 제외)
![](/main/cap_06.gif)
